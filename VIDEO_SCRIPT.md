## 🎬 Comandos para Captura via CLI (Alternativa)

### Usando FFmpeg (capturas automáticas):
```bash
# Capturar tela por 90 segundos
ffmpeg -f x11grab -video_size 1920x1080 -framerate 60 -i :0.0 -t 90 -c:v libx264 -preset ultrafast crm-demo.mp4

# Converter para GIF (preview)
ffmpeg -i crm-demo.mp4 -vf "fps=15,scale=800:-1" -loop 0 crm-preview.gif
```

### Usando Playwright (automação):
```javascript
// Gravar navegação automaticamente
const { chromium } = require('playwright');

(async () => {
  const browser = await chromium.launch();
  const context = await browser.newContext({
    viewport: { width: 1920, height: 1080 },
    recordVideo: { dir: 'videos/' }
  });
  const page = await context.newPage();
  
  await page.goto('https://tofariasti.github.io/crm/');
  await page.waitForTimeout(3000);
  
  // Navegar pelas seções
  await page.click('text=Leads');
  await page.waitForTimeout(2000);
  
  await page.click('text=Vendas');
  await page.waitForTimeout(2000);
  
  await page.click('text=Clientes');
  await page.waitForTimeout(2000);
  
  await page.click('text=Relatórios');
  await page.waitForTimeout(2000);
  
  await context.close();
})();
```
